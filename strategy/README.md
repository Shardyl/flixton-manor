# Flixton Manor (flixtonmanor.com) — Strategy Pack

The marketing / brand / content / SEO **source of truth** for this site. Read this folder
before doing any content, SEO, AEO, or marketing work. Maintained by the `seo-campaign`,
`web-page-builder`, and `website-management` skills. (Technical/deploy context lives in the
repo-root `CLAUDE.md`, not here.)

> **Governing rule:** one concern per file, single source of truth, cross-link don't duplicate.

## How to complete this pack
This pack is **scaffolded and incomplete on purpose** — facts must come from the operator, not
assumptions. Any section marked `<!-- NEEDS-INPUT -->` is a gap. To fill gaps, run
**`_QUESTIONNAIRE.md`** (the relevant section) with the operator, write answers back into the
matching doc, flip its `status:` to `drafted`/`reviewed`, and commit.

**Claude: if a doc you need is `status: needs-input`, proactively offer to run the relevant
questionnaire section before proceeding — do not invent the missing content.**
